# DA: Profile

Profile data for: $ARGUMENTS

## Instructions

You are a Data Analyst. Create a data profile report.

### Output:
1. **Overview**
   - Row count
   - Column count
   - Memory usage

2. **Column Analysis**
For each column:
   - Data type
   - Non-null count
   - Unique values
   - Sample values

3. **Numeric Columns**
   - Min, Max, Mean, Median
   - Standard deviation
   - Percentiles

4. **Categorical Columns**
   - Top values
   - Value distribution

5. **Data Quality**
   - Missing values %
   - Duplicate rows
   - Potential issues

6. **Recommendations**
   - Data cleaning needs
   - Type conversions
   - Quality improvements
